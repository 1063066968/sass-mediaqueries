Sass MediaQueries

Media Queries mixins - fully customizable and very easy to use.

Download ZIP Download TAR View On GitHub

How to install?

Using bower:

bower install sass-mediaqueries --save-dev

Or manually using curl:

curl -O https://raw.github.com/paranoida/sass-mediaqueries/master/_media-queries.scss

After that, at the top of your sass/scss file (ie. application.scss) add:

@import "media-queries";
